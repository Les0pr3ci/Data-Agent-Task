A Databricks Data Agent that analyzes user profiles and viewership data to answer natural language questions, uncover viewing trends, generate insights, and support data-driven decision-making.

# 📊 Viewership & User Profiles Data Agent

## Overview
This repository contains a Databricks Data Agent built to analyze user profile and viewership datasets. The agent allows users to ask questions in natural language and receive meaningful insights without writing SQL queries.

The project demonstrates how AI-powered data agents can simplify data exploration and support data-driven decision-making.

## Dataset
The project uses two datasets:

- **User Profiles** – Contains demographic and user-related information.
- **Viewership** – Contains viewing activity and engagement data.

## Features
- Natural language querying
- Viewership trend analysis
- User profile analysis
- Audience insights
- Data-driven decision support
- AI-powered analytics using Databricks

## Technologies Used
- Databricks
- Databricks AI/Genie Data Agent
- SQL
- GitHub

## Example Questions
The data agent can answer questions such as:

-How many unique subscribers are in the User Profile dataset?
-How many viewing sessions are recorded in the Viewership dataset
-Which channel has the highest total watch time?
-Which gender watches the most content?
-What are the top five most-watched channels?
-Which hour of the day has the highest viewing activity?
